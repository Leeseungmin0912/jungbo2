# Security Log Analyzer

간단한 로그 분석과 Windows Security 로그 추출 기능을 제공하는 Python 기반 보안 분석 프로그램입니다.

## 프로젝트 소개

Security Log Analyzer는 사용자가 로그 파일을 직접 불러와 이상 행위를 탐지하고, 위험 점수와 의심 IP를 확인할 수 있도록 만든 보안 분석 도구입니다.

기존의 단순 로그 분석 기능뿐 아니라, Windows의 Security 이벤트 로그를 앱 내부에서 CSV 파일로 저장할 수 있는 기능도 추가하여 사용 편의성을 높였습니다.

이 프로젝트는 정보보안 분야를 공부하면서 로그 분석, 이벤트 탐지, 보고서 생성, 보안 도구 UI 구현 경험을 쌓기 위해 제작했습니다.

---

## 제작 목적

- 사용자가 로그 파일을 직접 분석하며 보안 이상 징후를 확인할 수 있도록 하기 위해 제작했습니다.
- 정보보안 분야에서 중요한 로그 분석과 이벤트 탐지 과정을 직접 구현해보기 위해 제작했습니다.
- Python과 Tkinter를 활용해 보안 도구 형태의 프로그램을 만들고 포트폴리오로 활용하고자 했습니다.
- Windows Security 로그를 앱 내부에서 추출할 수 있도록 하여 실제 보안 분석 흐름을 경험해보기 위해 제작했습니다.

## 주요 기능

### 1. 로그 파일 선택 및 분석
- `.log`, `.txt`, `.csv` 형식의 로그 파일 선택 가능
- 선택한 로그 파일을 분석하여 이상 행위 탐지

### 2. 위험 행위 탐지
다음과 같은 패턴을 기준으로 로그를 분석합니다.

- 로그인 실패 반복
- 민감 경로 접근 (`/admin`, `/wp-admin`, `/login`, `/config`)
- 404 오류 반복
- 500 오류 반복
- 야간 시간대 접속
- 과다 요청 발생

### 3. 위험 점수 및 위험 등급 표시
- 분석 결과를 바탕으로 위험 점수를 계산
- 위험도를 `낮음`, `주의`, `높음`, `매우 높음`으로 구분하여 표시

### 4. 의심 IP 목록 출력
- 탐지된 이상 행위와 관련된 IP를 별도로 정리
- 의심 IP를 선택하면 상세 통계 확인 가능

### 5. IP 상세 통계 확인
선택한 IP에 대해 아래 항목을 확인할 수 있습니다.

- 총 요청 수
- 로그인 실패 수
- 민감 경로 접근 수
- 404 발생 수
- 500 발생 수
- 야간 접속 수

### 6. 보고서 저장 기능
분석 결과를 다음 형식으로 저장할 수 있습니다.

- TXT 보고서
- HTML 보고서

### 7. 샘플 로그 생성 기능
- 테스트용 샘플 로그 파일 생성 가능
- 프로그램 기능 확인 및 시연용으로 활용 가능

### 8. Windows Security 로그 CSV 저장 기능
- 앱 내부에서 Windows Security 이벤트 로그를 CSV 파일로 저장 가능
- PowerShell을 직접 입력하지 않고 버튼 클릭으로 추출 가능

## 분석 기준 예시

### 로그인 실패 반복
동일 IP에서 로그인 실패가 5회 이상 발생하면 위험으로 판단합니다.

### 민감 경로 접근
`/admin`, `/wp-admin`, `/login`, `/config` 경로 접근이 감지되면 주의 대상으로 판단합니다.

### 404 오류 반복
동일 IP에서 404 오류가 3회 이상 발생하면 비정상 탐색 가능성이 있다고 판단합니다.

### 500 오류 반복
동일 IP에서 500 오류가 2회 이상 발생하면 위험으로 판단합니다.

### 야간 접속
새벽 0시부터 5시 사이의 접속이 3회 이상 발생하면 주의 대상으로 판단합니다.

### 과다 요청
동일 IP에서 요청이 20회 이상 발생하면 과다 요청으로 판단합니다.

## Windows Security 로그 추출 기능

프로그램에는 Windows Security 이벤트 로그를 CSV로 저장하는 기능이 포함되어 있습니다.

### 기능 설명
- 버튼 클릭으로 Security 로그를 CSV 파일로 저장
- 사용자가 저장 경로를 직접 선택 가능
- 저장 후 바로 파일 열기 가능

### 주의사항
- 이 기능은 Windows 환경에서 동작합니다.
- Security 로그 접근 시 관리자 권한이 필요할 수 있습니다.
- 현재 버전에서는 Windows Security CSV를 저장하는 기능만 포함되어 있으며,
  프로그램의 기본 로그 분석 기능은 별도의 웹/서버 로그 형식을 기준으로 작동합니다.
