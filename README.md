# Security Log Analyzer

Windows 보안 이벤트 로그를 분석하여 로그인 흔적, 관리자 권한 부여, 자격 증명 관련 이벤트 등을 확인할 수 있는 Python 기반 보안 로그 분석 도구입니다.  
보안 로그 요약, 의심 이벤트 확인, 보고서 저장 기능을 통해 사용자가 자신의 시스템 보안 상태를 직접 점검할 수 있도록 제작했습니다.

> Windows Security 로그를 쉽고 빠르게 분석할 수 있는 보안 로그 분석 프로그램

---

## 프로젝트 소개

Security Log Analyzer는 Windows Security 로그를 분석하여  
로그인 성공 이벤트, 관리자 권한 부여 이벤트, 자격 증명 관련 이벤트 등의 발생 내역을 확인할 수 있도록 만든 프로그램입니다.

정보보안 분야를 공부하면서 단순한 이론 학습에 그치지 않고,  
실제 보안 로그를 직접 분석할 수 있는 프로그램을 구현해보며 실습 경험을 쌓고 포트폴리오로 활용하기 위해 제작했습니다.

이 프로젝트는 단순히 로그를 나열하는 것이 아니라,  
이벤트 ID별 발생 횟수와 시간대를 정리하고 사용자가 의심스러운 흔적을 직접 판단할 수 있도록 돕는 데 목적이 있습니다.

---

## 제작 목적

- Windows 보안 로그를 직접 분석할 수 있는 도구를 만들기 위해
- 정보보안 분야에서 배우는 로그 분석 개념을 실제 프로그램으로 구현해보기 위해
- 로그인 흔적, 권한 상승, 자격 증명 관련 이벤트를 직접 확인할 수 있도록 하기 위해
- 추후 포트폴리오와 자기소개서, 면접 자료로 활용할 수 있는 프로젝트를 만들기 위해

## 주요 기능

### 1. Windows 보안 로그 분석
- Windows Security 로그를 CSV 형태로 저장 및 분석
- 주요 이벤트 ID를 기준으로 보안 이벤트를 분류
- 반복적으로 발생한 이벤트를 요약하여 확인 가능

**주요 분석 이벤트**
- **4624** : 로그인 성공
- **4672** : 관리자 권한이 포함된 로그인
- **5379** : 자격 증명 관련 정보 조회

### 2. 이벤트 발생 횟수 및 시간대 요약
- 이벤트 ID별 발생 횟수 확인
- 최초 발생 시간과 마지막 발생 시간 확인
- 최근 발생 시간 목록 출력 가능

### 3. 의심 로그 확인
- 사용자가 PC를 사용하지 않은 시간대의 로그인 관련 이벤트 확인
- 비정상적인 접근으로 보일 수 있는 흔적을 점검 가능
- 덮개 열림, 절전 복귀, 자동 로그인, 시스템 서비스 실행 등 정상 가능성도 함께 고려할 수 있도록 설계

### 4. 보고서 저장 기능
- 분석 결과를 텍스트 또는 보고서 형태로 저장 가능
- 필요 시 점검 기록으로 남겨 포트폴리오나 보안 점검 자료로 활용 가능

### 5. GUI 기반 인터페이스
- Tkinter 기반 GUI 제공
- 버튼 클릭으로 로그 분석 가능
- 로그 창에서 분석 결과를 쉽게 확인 가능

## 사용 기술

- **Python**
- **Tkinter** : GUI 구성
- **csv / os / datetime / re** : 로그 처리 및 파일 저장
- **subprocess / PowerShell 연동** : Windows 보안 로그 추출
- **PyInstaller** : exe 파일 배포

## 프로그램 흐름

1. Windows Security 로그를 추출
2. CSV 파일로 저장
3. 이벤트 ID 기준으로 로그 분석
4. 이벤트 발생 횟수 및 시간대 요약
5. 의심 가능한 이벤트 확인
6. 결과를 보고서로 저장
